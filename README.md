## To run locally:

- Install Gatsby CLI

`npm install -g gatsby-cli`

- Install required dependencies

`npm install`

- Start dev server

`npm start`

## To deploy to prod:

- Generate a static build

`npm run build`

- Preview static build in your browser

`npm run serve`
