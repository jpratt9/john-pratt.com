  • Echelon: A Spotify Jukebox

    Echelon: A Spotify Jukebox

    Echelon is an Android and web app that allows a group of people to connect and queue up music together. The leader can stream that queue for the whole group. Music can be voted on in order to change it's order in the queue, making sure the best music is playing all the time.

    Tech

    The Android app is using Spotify's Android SDK to manage Spotify authentication and music streaming. The webapp is using Angular to dynamically load and show content. Both the Android app and webapp use the same Firebase backend. The backend is authenticated via the Spotify login or anonymous authentication. Firebase allows for realtime data updates which greatly improves the experience over a traditional REST api.

    Android Alpha Webapp
